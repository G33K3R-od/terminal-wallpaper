ТЕКСТ ДЛЯ МАСТЕРСКОЙ STEAM
==========================

Terminal Dashboard — терминальные обои: часы, погода, таймеры, аудио-визуализация, медиа (Spotify / браузер через WE).

Работает сразу после подписки — ничего скачивать не нужно.

ОПЦИОНАЛЬНО (Windows):
В папке обоев есть setup.bat — один запуск добавляет CPU / RAM / GPU / диски. Не обязателен.

---

EN: Works out of the box. Optional: run setup.bat once (Windows) for live PC stats.


КАК ВЫЛОЖИТЬ В МАСТЕРСКУЮ (пошагово)
=====================================

ПОДГОТОВКА ПАПКИ (один раз перед загрузкой)

  1) Удали из папки terminal-wallpaper (не обязательны подписчикам):
     - stats.json          (твои личные данные ПК)
     - .stats-autostart-installed   (служебный маркер)

  2) Оставь setup.bat, collect-stats.ps1 и т.д. — это опциональный бонус, в описании так и пиши.

  3) Превью для Steam:
     - Открой обои в редакторе WE → меню → сделать превью / snapshot
     - Должен появиться preview.gif (или preview.jpg) в папке проекта
     - В project.json уже указано: "preview": "preview.gif"


ЗАГРУЗКА В WORKSHOP

  1) Steam → Wallpaper Engine → запустить.

  2) Найди «Terminal Dashboard» в «Созданные обои» / myprojects.

  3) ПКМ по обоям → «Поделиться в Мастерской» /
     Share wallpaper on Workshop
     (или: открыть в редакторе → кнопка «Опубликовать» / Workshop).

  4) Заполни:
     - Название: Terminal Dashboard
     - Описание: текст из начала этого файла (RU + EN)
     - Теги: Technology, Customizable, Clock, Audio
     - Возраст: Everyone (уже в project.json)
     - Видимость: Public / Friends — как хочешь

  5) Дождись загрузки → Steam проверит → появится в Workshop.

  6) После публикации обновления: снова ПКМ → «Обновить в Мастерской» /
     Update on Workshop (если менял файлы).


ВАЖНО

  • Публиковать нужно из папки, которую WE уже видит как проект
    (myprojects\terminal-wallpaper), не копировать откуда попало.

  • У подписчиков без setup.bat обои всё равно нормальные (автономный режим).

  • Не пиши в описании «обязательно запустите скрипт» — только «опционально».

  • В WE: Настройки → Общие — у подписчиков должна быть включена
    интеграция с медиа (для блока Music).


ЕСЛИ НЕТ ПУНКТА «ПОДЕЛИТЬСЯ»

  • Убедись, что ты автор (свой myprojects, не подписка на чужие обои).
  • Запусти WE от имени обычного пользователя, Steam залогинен.
  • Обнови Wallpaper Engine.
  • Альтернатива: редактор обоев → File / Share → Workshop.
